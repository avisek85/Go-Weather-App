# Go-Weather-App
## How to run
First, Clone this repo 
then
pull docker image
```
docker pull sachinparihar/weather-app:latest
```
run the docker image
```
docker run -p 8080:8080 sachinparihar/weather-app:latest
```
browser the url ```localhost:8080/ ```
